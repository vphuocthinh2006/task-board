# CS221 Task Board 🤖

Bảng quản lý công việc dạng **Kanban Board** cho dự án CS221 — Hệ thống Agent Chatbot Đa Tác Vụ.

## 🌐 Demo

👉 [https://vphuocthinh2006.github.io/to-do-task/](https://vphuocthinh2006.github.io/to-do-task/)

## ✨ Tính năng

- **4 cột Kanban**: To Do → In Progress → Review → Done
- **Drag & Drop** kéo thả task giữa các cột
- **25 tasks mẫu** phân bổ cho 4 thành viên
- **Thêm / Sửa / Xóa** task
- **Bộ lọc** theo người & module
- **localStorage** — dữ liệu tự lưu trên trình duyệt
- **Dark glassmorphism theme** 🌙
- **Responsive** — tương thích mobile
- **Phím tắt**: `N` = thêm task, `Esc` = đóng modal

## 🛠️ Tech Stack

- HTML5 + CSS3 + JavaScript (Vanilla)
- Google Fonts (Be Vietnam Pro)
- localStorage cho data persistence

## 🚀 Cách chạy

Mở file `index.html` trên trình duyệt, hoặc truy cập link demo ở trên.
